# Multimodal Driver State Detection for Physiological Signals

## Overview
This project is an AI-powered driver state detection system designed to improve road safety by monitoring the driver's condition in real time.

## Features
- Facial expression recognition
- Driver drowsiness detection
- Stress and emotion monitoring
- Real-time alerts
- Computer vision based analysis

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy
